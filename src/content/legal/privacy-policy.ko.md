## 1. 개요

ZeroPamine("회사" 또는 "서비스")은 사용자의 개인정보를 소중히 여기며, 「개인정보 보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」, EU GDPR(General Data Protection Regulation), 미국 CCPA(California Consumer Privacy Act) 등 관련 법규를 준수합니다.

본 개인정보 처리방침은 ZeroPamine 모바일 애플리케이션 및 웹사이트(zeropamine.com)에 적용되며, 회사가 수집하는 개인정보의 항목, 수집 및 이용 목적, 보관 기간, 제3자 제공 내역 등을 안내합니다.

## 2. 수집하는 개인정보

### 2.1 계정 정보

**이메일/비밀번호 가입**
- 이메일 주소
- 비밀번호 (암호화되어 저장)
- 사용자 고유 식별자(UID)
- 표시 이름 (선택사항)
- 이메일 인증 상태

**Google 계정 로그인**
- Google 계정 이메일
- Google 계정 이름
- Google OAuth 토큰
- 사용자 고유 식별자(UID)

**Apple 계정 로그인**
- Apple 계정 이메일
- Apple 계정 이름 (선택사항)
- Apple OAuth 토큰
- 사용자 고유 식별자(UID)

**타이머 세션 기록**

- 계획된 시간 및 실제 소요 시간
- 완료 여부
- 사용자 노트 (선택사항)
- 연결된 루틴 정보

**앱 설정**
- 선택한 테마 (Coffee, Hourglass, Candle, Battery, Cat, Nyancat, Campfire, Tree)
- 자동 시작 설정
- 사운드 활성화 여부
- 사운드 볼륨 및 유형
- 언어 설정 (한국어/영어/일본어)

**루틴 구성**
- 사용자가 생성한 루틴 항목
- 루틴 이름, 시간, 순서

### 2.3 구매 정보

- 구매 상태 (활성/비활성)
- 구매 일시
- 구매 플랫폼 (Apple/Google)
- 제품 ID
- RevenueCat 고객 ID
- 마지막 검증 일시

### 2.4 기기 및 기술 정보

### 2.5 웹사이트 방문 정보 (Google Analytics)

## 5. 개인정보의 제3자 제공 및 공유

회사는 사용자의 개인정보를 다음과 같이 제3자와 공유합니다:

### 5.1 Firebase (Google Cloud Platform)
**제공 목적**: 인증, 클라우드 데이터베이스, 분석
**제공 항목**: 계정 정보, 세션 기록, 설정, 루틴, Pro 상태, 특수 테마 구매 정보, 기기 정보
**보유 기간**: 계정 삭제 시까지 (세션 기록은 90일)
**개인정보 처리방침**: https://firebase.google.com/support/privacy

**사용 서비스**:
- Firebase Authentication: 이메일/OAuth 인증
- Cloud Firestore: 클라우드 데이터 저장 및 동기화
- Firebase Analytics: 앱 사용 분석

### 5.2 RevenueCat
**제공 목적**: 인앱 구매 관리 및 처리
**제공 항목**: 사용자 UID, Pro 구독 정보, 특수 테마 구매 정보, 구매 내역
**보유 기간**: 법정 보유 기간 (세법에 따라 5년)
**개인정보 처리방침**: https://www.revenuecat.com/privacy

### 5.3 Google Sign-In
**제공 목적**: Google 계정으로 로그인
**제공 항목**: Google OAuth 토큰, 이메일, 이름
**보유 기간**: 계정 삭제 시까지
**개인정보 처리방침**: https://policies.google.com/privacy

### 5.4 Apple Sign-In
**제공 목적**: Apple 계정으로 로그인
**제공 항목**: Apple OAuth 토큰, 이메일, 이름 (선택사항)
**보유 기간**: 계정 삭제 시까지
**개인정보 처리방침**: https://www.apple.com/legal/privacy/

### 5.5 Google AdMob
**제공 목적**: 광고 표시 (비프로 사용자)
**제공 항목**: 광고 식별자, 기기 정보, IP 주소
**보유 기간**: Google AdMob 정책에 따름
**개인정보 처리방침**: https://policies.google.com/privacy

### 5.6 Google Analytics
**제공 목적**: 웹사이트 방문 분석
**제공 항목**: 익명화된 IP 주소, 브라우저 정보, 방문 페이지
**보유 기간**: 26개월 (Google Analytics 기본 설정)
**개인정보 처리방침**: https://policies.google.com/privacy

### 5.7 법적 요구사항에 따른 공개
회사는 다음의 경우 법적 의무에 따라 개인정보를 공개할 수 있습니다:
- 법원의 명령 또는 영장
- 법 집행 기관의 요청
- 사용자 또는 제3자의 생명, 신체, 재산 보호를 위해 필요한 경우

**중요**: 회사는 사용자의 개인정보를 마케팅 목적으로 제3자에게 판매하지 않습니다.

## 6. 개인정보의 보관 기간

### 6.1 계정 정보
- **보관 기간**: 계정 삭제 시까지
- **삭제 방법**: 앱 내 설정 > 계정 삭제

### 6.2 세션 기록
- **클라우드 보관**: 계정 삭제 시까지 무제한 보관
- **로컬 보관**: 최대 1,000개 세션 (기기에 저장)
- **조회 제한**: 앱에서는 최근 90일, 최대 500개 세션만 조회 가능
- **삭제**: 계정 삭제 시 모든 세션 기록 영구 삭제

### 6.3 설정 및 루틴
- **보관 기간**: 계정 삭제 시까지
- **로그아웃 시**: 로컬 데이터 삭제 (클라우드 데이터 유지)

### 6.4 구매 정보
- **보관 기간**: 구매 후 5년 (세법에 따른 법정 보유)
- **적용 대상**: ZeroPamine Pro 구매, 특수 테마 구매 (Campfire, Tree 등)

### 6.5 로컬 데이터 (기기 저장)
- **삭제 시점**: 로그아웃 또는 앱 삭제 시 즉시 삭제

### 6.6 백업 데이터
- **보관 기간**: 계정 삭제 후 30일 이내 완전 삭제

## 7. 개인정보의 국외 이전

### 7.1 데이터 저장 위치
사용자의 개인정보는 Google Cloud Platform(Firebase)을 통해 처리되며, 다음 국가의 서버에 저장될 수 있습니다:
- 미국
- 아시아 태평양 지역 (한국, 일본, 싱가포르 등)
- 유럽 연합

### 7.2 국외 이전 시 보호 조치
- **EU-US Data Privacy Framework**: GDPR 준수
- **Standard Contractual Clauses (SCC)**: EU 집행위원회 승인 표준 계약 조항
- **Firebase Security**: Google의 고도화된 보안 인프라

### 7.3 이전되는 정보
- 계정 정보 (이메일, UID)
- 세션 기록
- 설정 및 루틴
- 구독 정보
- 특수 테마 구매 정보

사용자는 서비스를 이용함으로써 위 국외 이전에 동의하는 것으로 간주됩니다.

## 8. 개인정보 보호를 위한 기술적/관리적 조치

### 8.1 기술적 조치
**암호화**
- 비밀번호: Firebase Authentication의 bcrypt 암호화
- 통신: HTTPS/TLS 1.3 암호화 프로토콜
- 저장: Firebase의 서버측 암호화 (AES-256)

**접근 통제**
- Firebase Security Rules를 통한 데이터 접근 제한
- 사용자별 UID 기반 데이터 격리
- OAuth 2.0 보안 프로토콜

**보안 인증**
- Google의 ISO 27001, SOC 2, SOC 3 인증

### 8.2 관리적 조치
- 개인정보 접근 권한 최소화
- 정기적인 보안 점검 및 업데이트
- 개인정보 처리 방침 교육

## 9. 쿠키 및 추적 기술

### 9.1 사용하는 쿠키
**웹사이트 (Google Analytics)**
- `_ga`: 사용자 식별 (2년)
- `_gid`: 세션 식별 (24시간)
- `_gat`: 요청 속도 제한 (1분)

**모바일 앱**
- Firebase Analytics SDK: 앱 사용 통계
- AdMob SDK: 광고 식별자 (비프로 사용자)

### 9.2 쿠키 관리 방법
**웹사이트**
- 브라우저 설정에서 쿠키 차단 또는 삭제 가능
- Google Analytics 거부: https://tools.google.com/dlpage/gaoptout

**모바일 앱**
- iOS: 설정 > 개인정보 보호 > 추적 > 앱이 추적을 요청하도록 허용 끄기
- Android: 설정 > Google > 광고 > 광고 개인 최적화 사용 중지

개인정보의 삭제를 요청할 수 있습니다.
- **방법**: 앱 내 설정 > 계정 삭제
- **효과**: 모든 클라우드 및 로컬 데이터 영구 삭제

### 10.4 개인정보 처리 정지권
개인정보 처리의 일시 정지를 요청할 수 있습니다.
- **방법**: zeropamine@appstdo.com으로 요청

### 10.5 개인정보 이동권 (GDPR)
개인정보를 다른 서비스로 이동할 수 있도록 데이터를 제공받을 수 있습니다.
- **방법**: zeropamine@appstdo.com으로 요청
- **형식**: JSON 또는 CSV 파일

### 10.6 동의 철회권
개인정보 수집 및 이용 동의를 철회할 수 있습니다.
- **방법**: 계정 삭제 또는 이메일 요청

### 10.7 권리 행사 방법
- **이메일**: zeropamine@appstdo.com
- **처리 기간**: 요청 후 30일 이내 (GDPR: 1개월 이내)
- **신원 확인**: 본인 확인을 위해 추가 정보를 요청할 수 있습니다

### 10.8 CCPA (캘리포니아 거주자) 권리
